# Construction Idle Game

Mobile idle construction management game developed using React Native and Expo.

## Features

- Idle income system
- Upgrade system
- Project lifecycle management
- Offline progress
- Physics-based interactions
- Persistent save/load

## Tech Stack

- React Native (Expo)
- TypeScript
- Zustand
- AsyncStorage
- Skia
- Matter.js

## Run locally

```bash
npm install
npx expo start
